# レポート日はプロフィールの一日開始時刻から導出する

- ステータス: 承認済み
- 日付: 2026-07-23

食事・運動・体重の実日時は変更せず、`config/profile.json` の `day_start_time` から
レポート日を読み取り時に導出する。開始時刻より前の記録を前日分へ含めることで、深夜の
夜食を前日の生活記録として扱いつつ、実際に記録した日時も保持する。未設定時は00:00とする。

## 検討した代替案

- **記録日時を前日へ変更する**: 集計は単純になるが、実際に食べた日時が失われる。
- **レポート日をDBへ保存する**: 読み取りは単純になるが、開始時刻を変更したときに既存記録の
  再計算が必要になり、日時と派生値がずれる可能性がある。
